## 업데이트 방법

### 바이너리 업데이트
```bash
# 1. 현재 실행 중인 노드 중지
bitcoin-cli stop

# 2. 새 버전 다운로드 및 설치
# (설치 가이드 참고)

# 3. 노드 재시작
bitcoind -daemon

# 4. 버전 확인
bitcoin-cli --version
```

### 소스 코드에서 빌드한 경우
```bash
# 1. 노드 중지
bitcoin-cli stop

# 2. 저장소 업데이트
cd bitcoin
git pull

# 3. 재빌드
./autogen.sh
./configure
make

# 4. 설치
sudo make install

# 5. 노드 재시작
bitcoind -daemon
```

## 주의사항

- 업데이트 전 항상 백업을 수행하세요
- 메인넷에서 테스트넷에서 먼저 테스트하는 것을 권장합니다
- 하위 호환성이 없는 업데이트의 경우 마이그레이션 가이드를 확인하세요
